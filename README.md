# L2 Studio - NHentai API

A library for [nhentai.net](https://nhentai.net) http web api

## 安装

在 `.npmrc` 文件写入下行配置：

```text
@l2studio:registry=https://npm.pkg.github.com
```

然后：

```shell
npm install @l2studio/nhentai-api

# or

yarn add @l2studio/nhentai-api
```
